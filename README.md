# 🛒 mini_OnShop

A simple **mini_OnShop Web Application** built using **HTML, CSS, JavaScript, PHP, and MySQL**.
This project allows users to browse products, add items to their shopping cart, and place orders through an easy-to-use web interface. It also includes an **admin panel** where administrators can manage products, categories, and orders.

---

## 🚀 Features

*  User Registration and Login
*  Browse Products by Category
*  Add Products to Shopping Cart
*  Update or Remove Items from Cart
*  Checkout and Place Orders
*  Admin Dashboard for Managing Products
*  Chatbox System
*  Responsive and User-Friendly Interface

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* PHP
* MySQL
* Bootstrap
* XAMPP / WAMP

## 📦 Installation

Follow these steps to run the project locally.

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/musfiq-21/OnlineShopping.git
```

### 2️⃣ Navigate to the Project Directory

```bash
cd OnlineShopping
```

### 3️⃣ Move Project to Local Server

Move the project folder into your local server directory.

For **XAMPP**

```
xampp/htdocs/
```

For **WAMP**

```
wamp/www/
```

---

## ▶️ Run the Application

1. Start **Apache** and **MySQL** from XAMPP/WAMP.
2. Open **phpMyAdmin** in your browser.
3. Create a new database named:

```
shopdb
```

4. Import the provided **SQL file** into the database.

---

## 🌐 How to Use

1. Make sure **Apache and MySQL** services are running.
2. Open your browser.
3. Visit the following URL:

```
http://localhost/mini_OnShop
```

4. Register a new account or log in.
5. Browse products and add items to the cart.
6. Proceed to checkout and place an order.

---
